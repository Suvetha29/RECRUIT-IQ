# RECRUIT-IQ

An AI-powered recruitment platform built with FastAPI and React that helps HR professionals create job postings using AI and manage applications efficiently.

## Features
- 🤖 AI-generated job descriptions using Groq LLM
- 👥 Role-based dashboards (HR & Candidates)
- 💼 Job posting management
- 📝 Application tracking system
- 🔐 Secure JWT authentication

## Tech Stack
**Backend:** Python-FastAPI, SQLAlchemy, LLM, JWT  
**Frontend:** React.js, React Router, Tailwind CSS 

## Setup
```bash
# Backend
cd backend
pip install -r requirements.txt
uvicorn main:app --reload

# Frontend
cd frontend
npm install
npm start
```
